# fusion_table()

Initiliazes Datatables.

---

fusion_table( string $table_id [, array $options ] ) : string

## Parameters

$table_id (string) (Required)

$options (array) (Optional) Default value: []

## Return Values

(string)

## Examples

```php
echo '<table id="'.fusion_table('test_table').'">....</table>';
```
